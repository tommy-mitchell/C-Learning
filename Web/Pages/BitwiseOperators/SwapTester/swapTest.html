Bitwise XOR Swap Tester

Overview

This program is a demonstration of the exclusive or (XOR) swap, which is

an algorithm that uses the exclusive or bitwise operation to swap the values of two variables without using the temporary variable which is normally required.

While efficient in theory, the normal algorithm is usually as fast or faster due to advances in modern CPUs. Perhaps one use of the XOR swap is in writing very-optimized assembly, when every byte of memory counts. Luckily, I'm not planning on doing that any time soon.
