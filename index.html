C Demos

Overview

A collection of C demos compiled into WASM and run with a minimal JS implementation of WASI. The source can be found here.

Demos

    Bitwise Operations

  1. Arithmetic Right Shift Test - Does a given machine perform an arithmetic or a logical right shift?
  2. Int Size Test - How big is an int on a given machine?
  3. Bit Rotation - Rotate the bits in an arbitrary number.
  4. Bit Pattern Search - Find a given series of bits in a number.
  5. Bitwise XOR Swap - Swap two given numbers with a bitwise XOR operator.
  6. Bit Manipulation - UNIMPLEMENTED.

    Project Euler

  1. Exercise 1 - Find the sum of all multiples of 3 and 5.
  2. Exercise 2 - Find the sum of all even Fibonacci numbers less than 4 million.

    Games

  1. 100 or Bust - UNIMPLEMENTED.
  2. Number Guesser - UNIMPLEMENTED.

    Miscellaneous

  1. Number to Word - Convert a given number into its textual representation.
